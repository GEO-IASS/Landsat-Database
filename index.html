About

This is a database manager to help organize Curtis Woodcock's remote sensing group at Boston University. The group has recently been analyzing time series models developed using entire Landsat image stacks to look at landcover change. The model used for analysis is termed CCDC (Continuous Change Detection and Classification) and was developed by Zhe Zhu and Curtis Woodcock (Zhu and Woodcock 2014). Recently, it has been used to create a near-indentical open-source Python port, YATSM (Yet Another Time Series Model), by Chris Holden.

More information on the can be found on their respective code repositories for CCDC and YATSM. In addition, the original repository containing an up-to-date data table containing the scenes used to create the map seen above can be found by clicking the View On Github link above.
